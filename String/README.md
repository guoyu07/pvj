###字符串
不逼逼了 直接上代码 喜欢python的字符串切片功能啊


来一个字符串

```python
viking = 'tattoo on my shoulder'
#各种切片玩
viking[0]
#'t'
viking[0:6]
#'tattoo'
viking[-8:]
#可以省略偶 很好用 'shoulder'
```
